================================================================================
KEY INFORMATION
================================================================================

API Key (Required):
- Get from: https://console.groq.com/
- Format: Starts with "gsk_"
- Put in: Create .env file with GROQ_API_KEY=gsk_xxx

Access Points (After Running):
- Frontend: http://localhost:3000
- Backend API: http://127.0.0.1:8000/api/
- Admin Panel: http://127.0.0.1:8000/admin

Virtual Environment (Important):
- macOS/Linux: source .venv/bin/activate
- Windows: .venv\Scripts\activate
- Deactivate: deactivate

Required Directories (Create before running):
- media/ (for uploaded PDFs)
- vector_db/ (for ChromaDB)
- chroma_db/ (for ChromaDB)
